рЃЏрЃЮрЃњрЃћрЃАрЃљрЃџрЃЏрЃћрЃЉрЃўрЃЌ, рЃћрЃА рЃљрЃарЃўрЃА рЃЕрЃЋрЃћрЃюрЃў рЃЎрЃБрЃарЃАрЃўрЃА рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў. рЃЋрЃўрЃгрЃДрЃћрЃЉрЃЌ рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃўрЃЌ.

## рЃарЃљ рЃљрЃарЃўрЃА HTML?
HTML - рЃћрЃА рЃљрЃЉрЃарЃћрЃЋрЃўрЃљрЃбрЃБрЃарЃљ рЃўрЃерЃўрЃцрЃарЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф HyperText Markup Language. HyperText рЃњрЃБрЃџрЃўрЃАрЃ«рЃЏрЃЮрЃЉрЃА рЃўрЃАрЃћрЃЌрЃў рЃбрЃћрЃЦрЃАрЃбрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃА рЃЊрЃљ HTML - рЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃАрЃљрЃА рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃў рЃўрЃДрЃЮ рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃўрЃюрЃЮрЃЋрЃљрЃфрЃўрЃљ рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃљрЃА рЃўрЃФрЃџрЃћрЃЮрЃЊрЃљ рЃарЃЮрЃЏ рЃАрЃ«рЃЋрЃљ рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљ рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃў рЃћрЃарЃЌрЃЏрЃљрЃюрЃћрЃЌрЃЌрЃљрЃю рЃЊрЃљрЃњрЃћрЃЎрЃљрЃЋрЃерЃўрЃарЃћрЃЉрЃўрЃюрЃљ. рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃљрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЊрЃўрЃЊрЃў рЃЊрЃарЃЮ рЃњрЃљрЃЋрЃўрЃЊрЃљ рЃЊрЃљ HTML - рЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃўрЃф рЃњрЃљрЃюрЃЋрЃўрЃЌрЃљрЃарЃЊрЃљ. рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃљрЃ«рЃљрЃџ рЃЋрЃћрЃарЃАрЃўрЃљрЃА HTML5 рЃ░рЃЦрЃЋрЃўрЃљ. HTML5 рЃЉрЃЏрЃБрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃарЃЊрЃљ рЃФрЃљрЃџрЃўрЃљрЃю рЃЉрЃћрЃЋрЃа рЃАрЃ«рЃЋрЃљ рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃА рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА. рЃљрЃЏ рЃЎрЃБрЃарЃАрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃЊрЃљрЃЋрЃцрЃљрЃарЃљрЃЋрЃЌ рЃДрЃЋрЃћрЃџрЃљ рЃўрЃЏ рЃАрЃљрЃГрЃўрЃарЃЮ рЃЌрЃћрЃЏрЃљрЃА рЃарЃљрЃф рЃЊрЃљрЃњрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ рЃљрЃЏ рЃћрЃюрЃўрЃА рЃЎрЃљрЃарЃњрЃљрЃЊ рЃерЃћрЃАрЃгрЃљрЃЋрЃџрЃўрЃАрЃЌрЃЋрЃўрЃА. 

рЃљрЃЦрЃЋрЃћ рЃўрЃБрЃюрЃЊрЃљ рЃљрЃЋрЃдрЃюрЃўрЃерЃюрЃЮрЃЌ рЃарЃЮрЃЏ HTML рЃљрЃа рЃљрЃарЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ. рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљ рЃАрЃљрЃўрЃбрЃќрЃћ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃБрЃџрЃў рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃўрЃА рЃерЃћрЃАрЃљрЃЦрЃЏрЃюрЃћрЃџрЃљрЃЊ. рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃљрЃАрЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљ [JavaScript](/courses/javascript) - рЃўрЃЌ рЃерЃћрЃАрЃљрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃАрЃБрЃарЃў рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃўрЃА рЃЎрЃћрЃЌрЃћрЃЉрЃљ. рЃљрЃЏ рЃћрЃюрЃўрЃА рЃЊрЃљрЃБрЃцрЃџрЃћрЃЉрЃљрЃф рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ BitCamp - рЃўрЃА рЃАрЃљрЃўрЃбрЃќрЃћ рЃњрЃљрЃюрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў [рЃерЃћрЃАрЃљрЃЉрЃљрЃЏрЃўрЃАрЃў рЃЎрЃБрЃарЃАрЃўрЃА](/courses/javascript) рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ. 

## рЃАрЃљрЃўрЃюрЃбрЃћрЃарЃћрЃАрЃЮ рЃцрЃљрЃЦрЃбрЃў
HTML рЃњрЃљрЃЏрЃЮрЃўрЃњрЃЮрЃюрЃљ Tim Berners-Lee - рЃўрЃЏ 1989 рЃгрЃћрЃџрЃА, CERN - рЃерЃў рЃЏрЃБрЃерЃљрЃЮрЃЉрЃўрЃАрЃљрЃА. рЃЏрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ рЃБрЃцрЃарЃЮ рЃЋрЃарЃфрЃџрЃљрЃЊ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљрЃЌ рЃгрЃљрЃўрЃЎрЃўрЃЌрЃ«рЃЮрЃЌ рЃЕрЃЋрЃћрЃюрЃА рЃЉрЃџрЃЮрЃњрЃќрЃћ - [рЃЋрЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃљ](/blog/vebis-istoria)


## HTML - рЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ
рЃЏрЃЮрЃЊрЃўрЃЌ рЃерЃћрЃЋрЃ«рЃћрЃЊрЃЮрЃЌ рЃЏрЃљрЃарЃбрЃўрЃЋрЃў HTML рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃА:
```html
<!DOCTYPE html>
<html>
<head>
    <title>рЃњрЃЋрЃћрЃарЃЊрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃў</title>
</head>
<body>
    <h1>рЃЕрЃћрЃЏрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃАрЃљрЃЌрЃљрЃБрЃарЃў</h1>
    <p>рЃЕрЃћрЃЏрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў HTML рЃљрЃЉрЃќрЃљрЃфрЃў.</p>
</body>
</html>
```

`<html>` рЃљрЃарЃўрЃА рЃўрЃА рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃЏрЃЌрЃћрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ. 

`<head>` рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃў рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ (meta) рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃљрЃЏ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃерЃћрЃАрЃљрЃ«рЃћрЃЉ. рЃљрЃЦ рЃљрЃарЃАрЃћрЃЉрЃБрЃџрЃў рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃљрЃа рЃЕрЃљрЃюрЃА рЃЋрЃћрЃЉ рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ. рЃЌрЃБрЃЏрЃфрЃљ рЃЏрЃљрЃАрЃерЃў рЃЏрЃЮрЃЌрЃљрЃЋрЃАрЃћрЃЉрЃБрЃџрЃў рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљрЃА рЃљрЃгрЃЋрЃЊрЃўрЃА рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃА. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ `<title>` рЃбрЃћрЃњрЃў рЃарЃЮрЃЏрЃћрЃџрЃерЃўрЃф рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃА рЃЋрЃгрЃћрЃарЃЌ рЃЊрЃљ рЃБрЃЏрЃћрЃбрЃћрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃћрЃЉрЃерЃў рЃљрЃЏ рЃАрЃљрЃЌрЃљрЃБрЃарЃА рЃЋрЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃўрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљрЃАрЃљ рЃљрЃю рЃбрЃљрЃЉрЃќрЃћ. 

`<body>` рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃўрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃЋрЃ«рЃћрЃЊрЃљрЃЋрЃЌ рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃќрЃћ. рЃЏрЃЌрЃћрЃџрЃў рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃ«рЃўрЃџрЃЋрЃљрЃЊрЃў рЃюрЃљрЃгрЃўрЃџрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ рЃљрЃЏ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃерЃў рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ. рЃАрЃљрЃЌрЃљрЃБрЃарЃў `<h1>`, рЃљрЃЉрЃќрЃљрЃфрЃў/рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃў `<p>` рЃЌрЃБ рЃАрЃ«рЃЋрЃљ рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃў рЃарЃЮрЃЏрЃџрЃћрЃЉрЃАрЃљрЃф рЃљрЃЏ рЃЎрЃБрЃарЃАрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃњрЃљрЃЋрЃћрЃфрЃюрЃЮрЃЉрЃўрЃЌ, рЃДрЃЋрЃћрЃџрЃљ рЃљрЃЦ рЃЏрЃЮрЃўрЃДрЃарЃўрЃА рЃЌрЃљрЃЋрЃА. 

рЃљрЃЏ рЃћрЃбрЃљрЃърЃќрЃћ рЃАрЃБрЃџ рЃћрЃА рЃљрЃарЃўрЃА, рЃерЃћрЃЎрЃЊрЃћрЃњ рЃЌрЃљрЃЋрЃерЃў рЃБрЃцрЃарЃЮ рЃЊрЃћрЃбрЃљрЃџрЃБрЃарЃљрЃЊ рЃњрЃљрЃюрЃЋрЃўрЃ«рЃўрЃџрЃљрЃЋрЃЌ HTML рЃћрЃџрЃћрЃЏрЃћрЃюрЃбрЃћрЃЉрЃА, рЃЏрЃљрЃЌ рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃАрЃљ рЃЊрЃљ рЃЌрЃЋрЃўрЃАрЃћрЃЉрЃћрЃЉрЃА. рЃўрЃЦрЃљрЃЏрЃЊрЃћ рЃЎрЃў рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃўрЃА рЃАрЃћрЃЦрЃфрЃўрЃљрЃА рЃЕрЃљрЃ«рЃћрЃЊрЃћ, рЃЏрЃЮрЃћрЃЏрЃќрЃљрЃЊрЃћ рЃЊрЃљ рЃфрЃЮрЃбрЃљ рЃўрЃЋрЃљрЃарЃ»рЃўрЃерЃћ рЃљрЃЏ рЃЌрЃљрЃЋрЃерЃў рЃюрЃљрЃАрЃгрЃљрЃЋрЃџрЃўрЃА рЃЏрЃћрЃбрЃљрЃЊ рЃњрЃљрЃљрЃќрЃарЃћрЃЉрЃљрЃерЃў. 


## рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃў
1. рЃњрЃљрЃЊрЃЏрЃЮрЃгрЃћрЃарЃћ рЃЊрЃљ рЃЊрЃљрЃљрЃўрЃюрЃАрЃбрЃљрЃџрЃўрЃарЃћ рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЎрЃЮрЃЊрЃўрЃА рЃћрЃЊрЃўрЃбрЃЮрЃарЃў. рЃЊрЃдрЃћрЃўрЃАрЃљрЃЌрЃЋрЃўрЃА рЃДрЃЋрЃћрЃџрЃљрЃќрЃћ рЃърЃЮрЃърЃБрЃџрЃљрЃарЃБрЃџрЃўрЃљ [Visual Studio Code](https://code.visualstudio.com/) - рЃФрЃљрЃџрЃўрЃљрЃю рЃЏрЃљрЃарЃбрЃўрЃЋрЃў рЃЊрЃљрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЊрЃљ рЃњрЃљрЃњрЃўрЃЏрЃљрЃарЃбрЃўрЃЋрЃћрЃЉрЃА HTML рЃЎрЃЮрЃЊрЃўрЃА рЃгрЃћрЃарЃљрЃА. рЃАрЃљрЃЏрЃЮрЃЏрЃљрЃЋрЃџрЃЮрЃЊ рЃЎрЃў рЃАрЃ«рЃЋрЃљ рЃћрЃюрЃћрЃЉрЃўрЃА рЃЊрЃљрЃБрЃцрЃџрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃерЃўрЃф рЃњрЃљрЃЏрЃЮрЃњрЃљрЃЊрЃњрЃћрЃЉрЃљ.
1. VS Code - рЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ, рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃўрЃА рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃћрЃюрЃў рЃљрЃ«рЃљрЃџрЃў HTML рЃцрЃљрЃўрЃџрЃў. рЃљрЃЏ рЃцрЃљрЃўрЃџрЃерЃў рЃЎрЃў рЃњрЃљрЃЊрЃљрЃљрЃЎрЃЮрЃърЃўрЃарЃћ рЃќрЃћрЃЏрЃЮрЃЌ рЃЏрЃЮрЃфрЃћрЃЏрЃБрЃџрЃў рЃЎрЃЮрЃЊрЃў. рЃњрЃљрЃ«рЃАрЃЮрЃЋрЃЊрЃћрЃА рЃарЃЮрЃЏ HTML рЃцрЃљрЃўрЃџрЃћрЃЉрЃА рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃЊрЃљрЃљрЃарЃЦрЃЋрЃљ, рЃЏрЃЌрЃљрЃЋрЃљрЃарЃўрЃљ рЃарЃЮрЃЏ `.html` рЃњрЃљрЃцрЃљрЃарЃЌрЃЮрЃћрЃЉрЃўрЃЌ рЃЉрЃЮрЃџрЃЮрЃЋрЃЊрЃћрЃЉрЃЮрЃЊрЃћрЃА. рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ `bitcamp.html`, `gamarjoba.html`, `app.html` рЃЊрЃљ рЃљ.рЃе. рЃљрЃЦрЃЋрЃћ рЃњрЃћрЃбрЃДрЃЋрЃў рЃарЃЮрЃЏ рЃ«рЃерЃўрЃарЃљрЃЊ рЃърЃўрЃарЃЋрЃћрЃџ HTML рЃцрЃљрЃўрЃџрЃА `index.html` = рЃА рЃљрЃарЃЦрЃЏрЃћрЃЋрЃћрЃю рЃ«рЃЮрЃџрЃЏрЃћ рЃарЃљрЃЊрЃњрЃљрЃю рЃАрЃўрЃбрЃДрЃЋрЃљ index рЃљрЃдрЃюрЃўрЃерЃюрЃљрЃЋрЃА рЃАрЃљрЃгрЃДрЃўрЃА/рЃЏрЃЌрЃљрЃЋрЃљрЃа рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃА. рЃЌрЃБрЃЏрЃфрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃЋрЃЌрЃЦрЃЋрЃўрЃЌ, рЃюрЃћрЃЉрЃўрЃАрЃЏрЃўрЃћрЃарЃў рЃАрЃљрЃ«рЃћрЃџрЃў рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃЮ. 
1. рЃњрЃљрЃ«рЃАрЃћрЃюрЃў рЃцрЃљрЃўрЃџрЃў рЃарЃЮрЃЏрЃћрЃџрЃўрЃЏрЃћ рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃўрЃЌ рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ Chrome рЃљрЃю Firefox рЃЊрЃљ рЃюрЃљрЃ«рЃћ рЃерЃћрЃЊрЃћрЃњрЃў. рЃњрЃўрЃџрЃЮрЃфрЃљрЃЋ ­ЪјЅ рЃћрЃА рЃерЃћрЃюрЃў рЃърЃўрЃарЃЋрЃћрЃџрЃў рЃЋрЃћрЃЉ-рЃњрЃЋрЃћрЃарЃЊрЃўрЃљ. 
1. рЃљрЃ«рЃџрЃљ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃерЃћрЃфрЃЋрЃљрЃџрЃЮ рЃЊрЃЮрЃЎрЃБрЃЏрЃћрЃюрЃбрЃўрЃА рЃАрЃљрЃЌрЃљрЃБрЃарЃерЃў рЃљрЃю рЃърЃљрЃарЃљрЃњрЃарЃљрЃцрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃбрЃћрЃЦрЃАрЃбрЃў рЃЊрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮ рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃљ. рЃерЃћрЃЏрЃЊрЃћрЃњ рЃЎрЃў рЃљрЃ«рЃџрЃўрЃЊрЃљрЃю рЃњрЃљрЃ«рЃАрЃюрЃљ рЃљрЃю рЃњрЃљрЃЊрЃљрЃбрЃЋрЃўрЃарЃЌрЃЮ/"рЃЊрЃљрЃљрЃарЃћрЃцрЃарЃћрЃерЃЮ" рЃЉрЃарЃЮрЃБрЃќрЃћрЃарЃўрЃА рЃцрЃљрЃюрЃ»рЃљрЃарЃљ. рЃ«рЃћрЃЊрЃљрЃЋ? рЃерЃћрЃюрЃў рЃфрЃЋрЃџрЃўрЃџрЃћрЃЉрЃћрЃЉрЃў рЃљрЃўрЃАрЃљрЃ«рЃљ. рЃюрЃћрЃџ-рЃюрЃћрЃџрЃљ рЃњрЃљрЃБрЃЌрЃљрЃЏрЃљрЃЏрЃЊрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃдрЃЏрЃЮрЃљрЃЕрЃћрЃю рЃарЃЮрЃЏ рЃЌрЃБрЃарЃЏрЃћ рЃљрЃарЃф рЃўрЃАрЃћ рЃФрЃюрЃћрЃџрЃў рЃДрЃЮрЃцрЃўрЃџрЃљ рЃћрЃА рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃў.
